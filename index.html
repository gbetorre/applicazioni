Server della Direzione Affari Istituzionali

at.univr.it è il server a disposizione della Direzione Affari Istituzionali. Esso ospita numerose applicazioni sviluppate internamente alla Direzione, come il software di mappatura e gestione dei rischi corruttivi (ROL) e il Registro dei Trattamenti dati personali.

RESPONSIVE LAYOUT

Tutte le nostre applicazioni web sono realizzate implementando Responsive Layout, in modo che le pagine siano fruibili senza problemi su qualsiasi dispositivo.

BOOTSTRAP 5.1.3

I nostri software utilizzano le versioni piú recenti delle piú moderne librerie standard, come bootstrap, jQuery, fontawesome, e relativi plugin.

BEST PRACTICE

I nostri software sono pensati per poter essere modulari, in modo che sia possibile riadattarli alla realtà di altri enti e pubbliche amministrazioni.

ROL [Rischi On Line]

ROL è il software che permette di ottenere la mappatura dei rischi corruttivi cui potenzialmente sono esposti i processi organizzativi.

È un'applicazione ad accesso riservato con vari ruoli e livelli di accesso.

È scritta in modo modulare e adattabile a tutte le organizzazioni.

Un clic, tanti calcoli

Le macchine devono fare i conti, mentre le persone fanno i ragionamenti.

Attraverso la risposta ad una serie di quesiti si ottiene in automatico il valore dei rischi.

Il software propone anche le misure piú adatte a ridurre i livelli di rischio.

TOL (Registro dei Trattamenti)

  • Applicazione ad accesso riservato
  • Multiruolo
  • Protocollo sicuro (https)
  • Database integrato
  • Funzioni di ricerca integrate
  • Stampa singolo trattamento
  • Visualizzazione HTML o PDF
  • Paginazione
  • Stampa registro annuale
  • Storicizzazione
  • Manutenzione evolutiva
  • Approccio modulare



Convenzioni Online di Direzione
